Before you start

Agent assurance codes for verification checks

A UK-regulated agent must carry out verification checks on all beneficial owners and managing officers no more than 3 months before the overseas entity is registered. They'll need to provide an agent assurance code to confirm they have authorisation to carry out verification checks.
