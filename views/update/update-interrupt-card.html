Before you start

Once an overseas entity is removed, its Overseas Entity ID will no longer be valid. It will not be able to buy, sell, transfer, lease or charge property or land in the UK, unless it re-registers.

Authentication code

You will need the overseas entity's authentication code to file this application. Find out how to get the authentication code if you don't already have it.

Check if any information needs to be updated

All information about the overseas entity, its beneficial owners and managing officers must be correct on the date of this application.

If you need to update any information, you can do that as part of this service. Any changes must have been verified by a UK-regulated agent no more than 3 months before this application is submitted.

Before you start

Authentication code

You will need the overseas entity's authentication code to file this update statement.

If you need to request one, search for the overseas entity on the Companies House register, then select 'Request an authentication code'. We'll send the code to the email address that we have on record for the overseas entity.

Verification checks

If there have been any changes to the overseas entity, its beneficial owners or managing officers, these changes must be verified by a UK-regulated agent. This must be done no more than 3 months before the date of this update statement.

The UK-regulated agent will need to provide their agent assurance code to confirm they have authorisation to carry out verification checks.

If you are not changing any information, you will not need to complete any verification checks.
